# Allocation driver

Allocation driver with tests

## Installing

Firstly, you have to build the driver and load it:

```
    make driver
    make load
```

## Running the tests

After that, build the tests and run them:

```
    make build
    make test
```

### Break down into end to end tests

After all, clean the folder and unload driver

```
    make unload
    make clean
```
